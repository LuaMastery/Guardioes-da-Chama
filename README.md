# Guardiões da Chama

Uma plataforma educacional interativa sobre filosofia de moderação e administração de comunidades online.

## Execução Local

**Pré-requisitos:**  Node.js


1. Instale as dependências:
   `npm install`
2. Configure a chave de API no arquivo [.env.local](.env.local)
3. Execute a aplicação:
   `npm run dev`
